83 - Strategic movement patterns maximize area coverage

85 Args:
86 player (Player): Player to conduct search with

89 Returns:

93 search_attempts = 0
94 max_attempts = min(player.map_width * player.map_height, 50)
95 while search_attempts < max_attempts:
96 tiles = MovementManager.look_around(player)
97 if len(tiles) > 0 and 'food' in tiles[0]:
98 NetworkManager.send_command(player, 'Take food')
99 for index, tile in enumerate(tiles):
100 if resource in tile:
101 return index
102 for _ in range(2):
103 MovementManager.move_forward(player)
104 search_attempts += 1
105 if search_attempts >= max_attempts:
106 break
107 MovementManager.turn_right(player)
108 search_attempts += 1
109 return -1
