Detailed Description

template<typename Derived>
class zappy::EntityBase< Derived >

CRTP (Curiously Recurring Template Pattern) base class for type-safe entities.

Template Parameters
DerivedThe derived entity class

EntityBase provides compile-time type identification for entity types using CRTP. This allows for efficient type checking without runtime overhead or virtual function calls.

The template automatically determines entity types based on the derived class, providing static type information that can be used for optimizations and type-safe operations.
