Detailed Description

Central manager for all user interface components.

The UIManager orchestrates all UI elements in the Zappy GUI application, including the HUD, various information panels, and debug interfaces. It handles layout management, event routing, and provides a unified interface for UI operations.

Key responsibilities:

  • Coordinate multiple UI panels and components
  • Handle window resize events and layout updates
  • Route input events to appropriate UI elements
  • Manage UI visibility and state
  • Provide notification and information display services

The manager uses a modular approach where each UI component is managed independently while being coordinated through this central system.
