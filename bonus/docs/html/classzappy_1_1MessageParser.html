Detailed Description

Parser for Zappy server protocol messages.

This class is responsible for parsing raw text messages received from the Zappy server into structured Message objects. It handles the complete server protocol, including map information, player actions, game events, and administrative messages.

The parser uses a command-based approach with registered handlers for each message type. It supports both single message parsing and buffer-based parsing for handling multiple messages received in a single network packet.

Key features:

  • Complete Zappy protocol support
  • Robust error handling and validation
  • Efficient parsing with registered handlers
  • Buffer management for partial messages
