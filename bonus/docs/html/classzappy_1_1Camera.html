Detailed Description

Advanced camera system for viewport management and navigation.

The Camera class provides a sophisticated camera system for the Zappy GUI, supporting multiple viewing modes, smooth movement, and various input methods. It handles viewport transformations, coordinate conversions, and provides intuitive navigation controls.

Key features:

  • Multiple camera modes (free, follow, team, overview)
  • Smooth interpolation for movement and zooming
  • Mouse and keyboard input handling
  • Coordinate system transformations
  • Boundary constraints and limits
  • Player and team tracking capabilities

The camera system uses smooth interpolation to provide fluid movement and zooming, enhancing the user experience.
