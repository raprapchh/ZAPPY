Detailed Description

@file resources.py
@brief Resource discovery and collection management for Zappy AI
@author Zappy Team
@date 2025

This module implements comprehensive resource discovery and collection strategies
for the Zappy AI system. It provides intelligent search algorithms, efficient
collection methods, and strategic resource gathering coordination.

The resource management system encompasses:
- Systematic resource discovery using vision-based scanning
- Intelligent search patterns for efficient map exploration
- Strategic resource collection with movement optimization
- Opportunistic food collection during resource searches
- Map-aware search boundaries and termination conditions

Key algorithmic features:
- Vision-based resource detection and tile indexing
- Spiral search patterns for comprehensive area coverage
- Map boundary respect to prevent infinite searching
- Multi-resource coordination for complex collection needs
- Fallback movement patterns when resources are scarce

The module coordinates closely with movement and network managers to
implement efficient resource collection while maintaining exploration
efficiency and preventing search loops or boundary violations.

Technical implementation:
- Tile-based vision system integration
- Dynamic search area calculation based on map dimensions
- Resource presence detection and navigation coordination
- Collection verification and success tracking
- Adaptive search termination based on map coverage
