Detailed Description

Manages the complete game world state for the Zappy simulation.

The World class is the central data model that maintains the entire state of the Zappy game world. It manages all game entities including players, teams, eggs, and the map itself with its tiles and resources.

Key features:

  • Maintains a grid-based map with resource distribution
  • Manages player lifecycles and positions
  • Handles team information and statistics
  • Tracks eggs and their states
  • Provides spatial queries for rendering optimization
  • Integrates with the event system for state changes

The world uses a normalized coordinate system where coordinates wrap around the map boundaries, creating a toroidal world topology as specified in the Zappy protocol.

See also
Tile for individual Map Management cell information
Player for Player Management entity management
Team for Team Management statistics and data
Egg for Egg Management lifecycle management
