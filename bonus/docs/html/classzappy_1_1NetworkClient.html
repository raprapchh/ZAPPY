Detailed Description

Asynchronous network client for communicating with the Zappy server.

This class provides a thread-safe, asynchronous interface for communicating with the Zappy server. It handles connection management, message parsing, and maintains connection statistics. The client operates on a separate thread to avoid blocking the main application loop.

Key features:

  • Asynchronous networking with dedicated thread
  • Thread-safe message queues for communication
  • Connection state management and monitoring
  • Automatic reconnection handling
  • Latency and bandwidth tracking
