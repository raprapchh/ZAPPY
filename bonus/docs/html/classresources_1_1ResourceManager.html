Detailed Description

Static class managing resource discovery and collection strategies.

This class implements intelligent resource gathering algorithms that
balance efficiency with thoroughness. It provides the core resource
collection functionality used throughout the AI system for both
survival needs and strategic progression.

Resource Discovery Strategy:
- Vision-based scanning for immediate resource detection
- Systematic movement patterns for area exploration
- Map-aware search boundaries to prevent infinite loops
- Opportunistic food collection during searches
- Adaptive termination based on map coverage

The class coordinates with movement and network managers to implement
efficient collection workflows while maintaining exploration patterns
that maximize resource discovery probability.

bool resources.ResourceManager.collect_resource ( Player player,
str resource )
static 
Execute complete resource collection workflow.

Implements the full resource acquisition process from discovery
through collection. This is the primary interface for resource
gathering used throughout the AI system.

Complete workflow:
1. Search for and locate target resource
2. Navigate to resource position
3. Execute Take command to collect resource
4. Verify collection success through server response

The method provides a single-call interface for resource collection
that handles all complexity internally while providing clear success
indication for strategic planning.

Success verification:
- Server "ok" response indicates successful collection
- Failed collection (resource taken, movement failed) returns False
- Network errors or resource absence returns False

Args:
    player (Player): Player to collect resource with
    resource (str): Name of resource to collect
    
Returns:
    bool: True if resource successfully collected, False otherwise

int resources.ResourceManager.search_for_resource ( Player player,
str resource )
static 
Search for specific resource using systematic exploration pattern.

Implements a comprehensive resource search algorithm that combines
vision-based detection with strategic movement patterns. The search
uses map dimensions to calculate optimal search boundaries and
prevents infinite searching in resource-sparse areas.

Search algorithm:
1. Vision scanning for immediate resource detection
2. Opportunistic food collection during searches
3. Systematic forward movement with periodic direction changes
4. Map-aware termination to prevent boundary violations
5. Right-turn pattern to ensure comprehensive coverage

Search optimization:
- Max attempts based on map size (prevents infinite loops)
- Capped at 50 attempts for performance reasons
- Opportunistic food collection maintains survival
- Strategic movement patterns maximize area coverage

Args:
    player (Player): Player to conduct search with
    resource (str): Target resource name to search for
    
Returns:
    int: Tile index of found resource (0-8+ based on vision),
         -1 if resource not found within search limits
