Detailed Description

Represents a player entity in the game world.

The Player class extends Entity to provide specific functionality for player characters. It manages player state including position, orientation, level, inventory, team affiliation, and various actions like incantation and broadcasting.

Key features:

  • Position tracking with smooth interpolation
  • Inventory management with resource counting
  • Team affiliation and color coding
  • Level progression tracking
  • Action state management (incantation, broadcasting)
  • Animation and visual representation support

Players are the primary interactive entities in the Zappy world and are responsible for resource collection, elevation rituals, and strategic gameplay.
