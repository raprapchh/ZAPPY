Detailed Description

Represents a player entity in the Zappy world with all relevant state.

This class encapsulates all information about a player including their
connection, position, level, team affiliation, and current AI state.
It serves as the central data structure for the AI decision-making process.

Attributes:
    socket: Network socket connection to the server
    level: Current evolution level (1-8)
    player_id: Unique identifier assigned by server
    map_width: Width of the game world
    map_height: Height of the game world  
    team_name: Name of the team this player belongs to
    state: Current AI behavioral state
    last_broadcast_time: Timestamp of last broadcast sent
    is_leader: Whether this player acts as team coordinator
