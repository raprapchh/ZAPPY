Detailed Description

Abstract base class for all game entities.

The Entity class provides the foundation for all interactive objects in the Zappy world. It defines the common interface and shared functionality that all entities must implement, including position management, visibility control, and update cycles.

Key features:

  • Pure virtual interface for polymorphic behavior
  • Position interpolation system for smooth movement
  • Visibility management
  • Type identification system
  • Bounding box support for spatial calculations

All entities in the game world (Players, Eggs, Resources) inherit from this base class and must implement the pure virtual methods to define their specific behavior.
