Detailed Description

Configuration data for connecting to and participating in a Zappy game.

Contains all necessary connection parameters and game settings
that the AI needs to join a specific game session.

Attributes:
    host: Server hostname or IP address
    port: Server port number
    team_name: Name of the team to join
