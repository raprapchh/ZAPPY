Detailed Description

Central configuration management class for the Zappy GUI application.

This class manages all configuration parameters for the GUI, including network settings, window properties, rendering options, and user interface preferences. It provides functionality to parse command-line arguments, load settings from files, and validate configuration parameters.

The configuration system supports both command-line argument parsing and file-based configuration loading, with validation to ensure all settings are within acceptable ranges.
