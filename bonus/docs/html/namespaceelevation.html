Detailed Description

@file elevation.py
@brief Level elevation and progression management for Zappy AI
@author Zappy Team
@date 2025

This module manages the complex process of player level elevation in the Zappy game.
It handles resource requirement calculation, collection coordination, and elevation
ritual execution. The system implements the complete elevation pipeline from
resource planning to ritual completion.

The elevation system encompasses:
- Comprehensive resource requirement definitions for all levels (1-8)
- Intelligent resource collection strategies with fallback patterns
- Strategic resource placement and elevation ritual execution
- Integration with survival systems to maintain food security during elevation
- Adaptive search patterns for efficient resource gathering

Key features:
- Level-specific resource requirement tables
- Progressive difficulty scaling with higher levels
- Intelligent resource prioritization and collection ordering
- Survival-aware resource gathering (food takes precedence)
- Robust error handling and retry mechanisms
- Strategic movement patterns for resource discovery

The module coordinates with resource, survival, and movement managers to
ensure successful elevation while maintaining player survival and efficiency.
