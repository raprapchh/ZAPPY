Detailed Description

Static class managing all aspects of player survival and food security.

This class implements a comprehensive food management system with multiple
priority levels to ensure player survival while optimizing resource allocation.
The system uses threshold-based decision making to balance immediate survival
needs with long-term strategic planning.

Food Management Thresholds:
- Critical (< 2): Emergency survival mode
- Low (< 5): Active food gathering required
- Warning (< 10): Proactive food collection recommended
- Safe (>= 15): Stockpiling target for higher levels

The class coordinates with movement and network managers to implement
efficient search patterns and resource collection strategies.
