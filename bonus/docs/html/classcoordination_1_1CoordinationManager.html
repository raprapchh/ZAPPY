Detailed Description

Static class managing team coordination and communication strategies.

This class implements intelligent team coordination through broadcast
messaging and AI state management. It balances effective communication
with spam prevention to maintain efficient team coordination.

Communication Strategy:
- Leaders broadcast more frequently (3s interval) for coordination
- Followers use longer intervals (5s) to reduce communication overhead
- Randomized timing prevents synchronized broadcast storms
- Duplicate message detection prevents repetitive communications
- State-based responses enable coordinated team behavior

The system promotes emergent team behavior through simple communication
protocols while avoiding the complexity of centralized coordination.
